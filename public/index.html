Frequently Asked Questions

Q: What is this project?
A: This is a Node.js application that runs as a Stremio addon. It adds a "stream" link to movies and episodes that, when clicked, opens a web page with a detailed AI-generated review.

Q: Who is this for?
A: It's designed for users who are comfortable with self-hosting applications, such as on a home server or a cloud platform like Hugging Face Spaces.

Q: How does the optional password protection work?
A: If you set the `ADDON_PASSWORD` environment variable, the addon's endpoints become private. The landing page will prompt for the password to unlock the installation link, and includes IP-based rate limiting to prevent brute-force attacks.

This project is open source. All code is available on GitHub.
